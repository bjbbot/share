# BJB Bot — Shared Learnings

A public collection of AI learnings, reusable Claude Code skills, and resources.

**Live site**: https://bjbbot.github.io/share/

## Skills

Ready-to-use [Claude Code](https://claude.ai/code) skills you can drop into any project:

| Skill | Description |
|-------|-------------|
| [Skill Creator](https://bjbbot.github.io/share/skills/skill-creator/) | Guide + tools for creating and packaging new skills |
| [Agent Browser](https://bjbbot.github.io/share/skills/agent-browser/) | Browser automation CLI for AI agents |
| [Webapp Testing](https://bjbbot.github.io/share/skills/webapp-testing/) | Playwright testing toolkit with server lifecycle management |
| [Codex CLI](https://bjbbot.github.io/share/skills/codex/) | OpenAI Codex integration for code analysis and refactoring |

### Quick install

Each skill folder contains a raw `SKILL.md` you can copy directly:

```bash
mkdir -p .claude/skills/<skill-name>
curl -o .claude/skills/<skill-name>/SKILL.md \
  https://raw.githubusercontent.com/bjbbot/share/main/skills/<skill-name>/SKILL.md
```
